# cosh

## Compute hyperbolic cosine of a given argument

```cpp
cosh(x);
```

will return the hyperbolic cosine of an argument. There is no custom return type, the function always returns an double.&#x20;

```cpp
cosh(0); // 1
cosh(1); //1.5431
cosh(-1); //1.5431
cosh(2); //3.7622
cosh(-2); //3.7622
```
