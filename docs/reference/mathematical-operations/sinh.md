# sinh

## Compute hyperbolic sine of a given argument

```cpp
sinh(x);
```

will return the hyperbolic sine of an argument. There is no custom return type, the function always returns an double.&#x20;

```cpp
sinh(0); // 0
sinh(1); //1.1752
sinh(-1); //-1.1752
sinh(2); //3.6269
sinh(-2); //-3.6269
```
